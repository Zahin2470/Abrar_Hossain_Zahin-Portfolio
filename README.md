## вңҰ About This Portfolio

A **premium, production-grade personal portfolio** built from scratch with Next.js 14, Tailwind CSS, and Framer Motion. Goes far beyond a standard portfolio вҖ” featuring a **live star canvas with spider-web cursor**, **dark/light mode with day sky and sun**, **lightning bolts**, **live GitHub stats dashboard**, **Kaggle live badge**, **interactive skill radar chart**, **4 free AI-powered tools** (chatbot, paper summarizer, research idea generator, resume builder), and much more.

> *"Turning data into intelligence, one model at a time."*

## вңҰ AI-Powered Tools (All Free В· No Account Needed)

All tools run on **Groq API (free tier)** with **Llama 3.3 70B** вҖ” no credit card, no signup required.

### рҹӨ– AI Chatbot (`/chat`)
Knows everything about Zahin вҖ” research papers, projects, skills, background. Visitors can ask anything and get instant, accurate answers.

### рҹ“„ Paper Summarizer (`/paper-summarizer`)
Paste any research paper abstract or title. Returns:
- **TL;DR** вҖ” one casual sentence
- **The Problem** вҖ” what gap it addresses
- **The Method** вҖ” specific techniques used
- **Key Findings** вҖ” results and numbers
- **Why It Matters** вҖ” real-world impact
- **Key Terms** вҖ” jargon decoded in plain English

### рҹ’Ў Research Idea Generator (`/research-ideas`)
Choose from **12 preset topic chips** (Medical Imaging, Cybersecurity, Green AI, NLP, Federated LearningвҖҰ) or type your own. Get 5 novel research directions each with:
- Specific title and punchy tagline
- Problem statement, methodology, novelty, expected outcome
- Difficulty level (MSc / PhD / Industry), time estimate, keywords

### рҹ“Ӣ Resume Builder (`/resume-builder`)
- **3 templates** вҖ” Sidebar, Classic, Minimal
- **5 colour themes** вҖ” Violet, Slate, Emerald, Rose, Indigo
- **Live preview** вҖ” updates as you type
- **PDF export** вҖ” one click, proper A4 formatting with colour printing
- **Sample data** вҖ” pre-fills with Zahin's real info as a reference

## вңҰ Quick Start

### 1 В· Create project & install

```bash

cd zahin-portfolio
npm install framer-motion
```

### 2 В· Copy all files

Replace the generated `src/` folder with the files from this repo.

### 3 В· Add your photo and resume


